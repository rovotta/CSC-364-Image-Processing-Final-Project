# CSC-364-Image-Processing-Final-Project
Pillow Python implementation of the BM3D image denoising algorithm (Dabov et al., 2007). Implements DCT/WHT transforms, block matching, 3D collaborative filtering with hard-thresholding and  Wiener filtering in two separate stages. Built for educational purposes to be presented at the 2026 Davidson College Verna Miller Case Symposium.
